# 📊 Vendor Performance Analysis

## 🔄 Data Pipeline

A data ingestion pipeline was built to efficiently load and manage data:

* Extracted data from multiple CSV files
* Loaded data into MySQL database for structured storage
* Used Python to automate data ingestion
* Ensured proper data types and schema creation in MySQL

### 🛠️ Pipeline Workflow

1. Read CSV files using Pandas
2. Performed basic cleaning and validation
3. Connected to MySQL using Python
4. Loaded data into chunks to respective tables

## 📝 Logging & Monitoring

A logging mechanism was implemented to track the data pipeline process:

* Recorded key steps such as data ingestion, transformation, and loading
* Captured errors and exceptions for debugging
* Maintained log files for monitoring pipeline execution


## 📌 Project Overview

This project focuses on analyzing vendor performance to identify inefficiencies in inventory management, pricing, and sales contribution. The goal is to generate actionable insights that can help improve profitability and optimize vendor relationships. Additionally, a data pipeline was developed to automate data ingestion from CSV files into a MySQL database, mimicking real-world data workflows.

---

## 🎯 Objectives

* Identify **low-performing vendors** based on sales and inventory turnover
* Analyze **stock movement and inventory efficiency**
* Evaluate **pricing and purchase patterns**
* Detect vendors contributing to **low profit margins**

---

## 📂 Dataset Description

The project uses multiple datasets:

* **Sales Data** – Contains transaction-level sales information
* **Purchases Data** – Records vendor purchase details
* **Purchase Prices** – Includes product pricing from vendors
* **Beginning Inventory** – Opening stock data
* **Ending Inventory** – Closing stock data

---

## 🛠️ Tools & Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn, sqlalchemy)
* Jupyter Notebook
* MYSQL
* Excel
* Power BI

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Merging multiple datasets for unified analysis
* Calculating **stock turnover ratio**
* Identifying **slow-moving inventory**
* Vendor-wise performance comparison
* Profit margin analysis

---

## 📈 Key Insights

* Optimize pricing for low-sales, high-margin products to improve sales volume
* Diversify vendors to reduce dependency and supply risks
* Use bulk purchasing to maintain competitive pricing and better inventory control
* Manage slow-moving inventory through optimized purchasing and clearance strategies
* Improve marketing and distribution for underperforming vendors

These actions can help increase profitability, reduce risks, and improve operational efficiency
